Usage Help

Deposit crypto funds

Deposit dash(coins)

1. In the "Puppi home" page, click on "Deposit dash(coins)".

Note: You cannot deposit fiat currency. Deposit other cryptos like bitcoin, ethereum, litecoin.

1. In the "Puppi home" page, click on "Deposit other coins".

Note: You cannot deposit fiat currency.

Usage

Getting started steps

1. Create a account using email-id or google signup.

2. Accept terms and conditions, Mention the country.

3. Give a complex password to the wallet.

3. Unlock the wallet after wallet is created.

4. Deposit funds and use the wallet for sending and receiving payment messages.

Doing a transaction example

1. Deposit 10$ worth of bitcoin or dash(coin) in "Puppi home" page.

2. Create a message with 1$ in "Puppi send" page.

3. Send the payment message to a friend through email, whatsapp.

3. Ask your friend to accept the payment in "Puppi direct" page.

Type of payment message to send

1. "Direct message", this message has PIN and Message included. Receiver can accept in one click.

2. "Part message", this message has PIN and Message separate. Receiver has to provide PIN to accept funds.

Type of payment message to receive

1. "Direct message", can be accepted in one click, using link provided.

2. "Part message", can be accepted after entering PIN along with Message.

Reverting payment message

1. The sender can revert the payment by going from "Puppi send" to the message. Then click on "Revert". This can be done before other user accepts the funds.
